# Epsilon Hydrology

Reproducible code and documentation for the epsilon-based catchment storage-discharge sensitivity study.

This repository is intended for release with the paper after submission/publication. It should contain only reproducible research materials:

- data acquisition and preprocessing scripts
- model training and inference code
- analysis notebooks or scripts
- environment files
- public documentation for reproducing figures and tables

Do not include meeting records, raw private notes, local demos, credentials, unpublished collaborator code without permission, or large raw ERA5-Land files.

Terminology: use `epsilon` as the primary target term. When needed, define `epsilon = GQ / Q`.

Main documentation:

```text
docs/SUMMARY.md
docs/TECHNICAL_METHODS.md
```
